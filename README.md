# 🧠 NLP Smart Feedback Analyzer

A smart feedback analysis system built using PHP, MySQL, HTML, and CSS that performs basic Natural Language Processing (NLP) to classify sentiment and extract meaningful keywords from user input.

---

## ✨ Features
- 😊 Sentiment Analysis (Positive, Negative, Neutral)  
- 🔑 Keyword Extraction from feedback  
- 📊 Feedback Count Dashboard  
- 💾 Stores data in MySQL database  

---

## 🛠️ Tech Stack
- 🌐 HTML, CSS  
- ⚙️ PHP  
- 🗄️ MySQL  

---

## 🚀 How to Run
1. Import the database in **phpMyAdmin**  
2. Place project folder inside `htdocs` (XAMPP)  
3. Start **Apache** and **MySQL**  
4. Open in browser:  
   http://localhost/feedback_project/index.php  

---

## 📌 Project Overview
This project takes user feedback as input and processes it using simple NLP techniques.  
It identifies whether the feedback is positive, negative, or neutral and also extracts important keywords for better understanding of user opinions.
